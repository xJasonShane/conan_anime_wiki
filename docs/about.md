# 关于

## 项目简介

名侦探柯南 Wiki 是一个专注于《名侦探柯南》电视动画作品的资料库网站,收录了所有剧集的详细信息,包括中文标题、日文标题、首播时间、对应漫画等内容。

## 功能特性

- 📊 **数据统计**: 提供剧集总数、年度分布等统计信息
- 📝 **剧集列表**: 完整的剧集列表,支持排序和分页
- 🔍 **搜索功能**: 快速搜索剧集号、标题等信息
- 📱 **响应式设计**: 完美适配桌面和移动设备

## 技术栈

- **框架**: VitePress
- **语言**: JavaScript (Vue 3)
- **样式**: CSS (Scoped Styles)
- **数据**: JSON

## 数据来源

本站数据来源于公开资料,仅供学习和参考使用。
